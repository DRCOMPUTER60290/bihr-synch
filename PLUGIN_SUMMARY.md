## ⚡ TÂCHES AUTOMATISÉES (WP-CRON) - LA CLÉ

Le plugin **automatise complètement** via WP-Cron:

```
┌─────────────────────────────────────────────┐
│ STOCKS                                      │
├─────────────────────────────────────────────┤
│ Synchronisation automatique BIHR → WooCommerce
│ • Toutes les heures              (fort trafic)
│ • 2× par jour            (matin 06:00 + soir)
│ • 1× par jour               (heure choisie)
│ • 1× par semaine              (dimanche)
│ ⚙️ Configuration: 1 dropdown
│ 🚀 Manuelle: Bouton "Maintenant"
└─────────────────────────────────────────────┘

┌─────────────────────────────────────────────┐
│ PRIX                                        │
├─────────────────────────────────────────────┤
│ Télécharge & applique Prices.csv BIHR
│ • Toutes les heures              (fort trafic)
│ • 2× par jour            (matin 06:00 + soir)
│ • 1× par jour               (heure choisie)
│ • 1× par semaine              (dimanche)
│ ⚙️ Configuration: 1 dropdown
│ 🚀 Manuelle: Bouton "Maintenant"
└─────────────────────────────────────────────┘
```
